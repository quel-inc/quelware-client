# quelware-client

Monorepo for QuEL system client libraries and tools.

## Packages

- [quelware-core](quelware-core/) — Shared entities and protobuf definitions
- [quelware-client](quelware-client/) — Client library for connecting to QuEL systems via gRPC
- [quelware-admin](quelware-admin/) — CLI tool for user management

## Documentation

Full documentation is published at <https://quel-inc.github.io/quelware-client/>:

- [Client](https://quel-inc.github.io/quelware-client/client/) — Python client library
- [Admin](https://quel-inc.github.io/quelware-client/admin/) — administrator CLI
- [Core](https://quel-inc.github.io/quelware-client/core/) — shared models and protobuf definitions

## Versioning

Each package is versioned independently. Tags follow the format `<package>/v<version>` (e.g., `quelware-client/v0.1.2`).
